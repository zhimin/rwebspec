Demo Project

This project demonstrates how easy you can develop automated web functional (acceptance) tests with iTest2.

Quick (in seconds) Start Guide

  1. Left mouse click 'google_spec.rb' on the left panel
  2. Left mouse click the blue triangle button to run!
  3. Sit back and view: your test running in an Internet explorer window
